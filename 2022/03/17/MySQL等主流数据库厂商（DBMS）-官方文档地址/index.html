三 SQL Server官方文档

SQL server有三个版本：

  1. 付费版，商（企）业版。

  2. Developer：Developer开发版，是一个全功能免费版本，许可在非生产环境下用作开发和测试数据库。

  3. Express：Express 是 SQL Server 的一个免费版本，非常适合用于桌面、Web 和小型服务器应用程序的开发和生产。

tips：将en-us换成zh-cn即可看到中文文档

SQL server文档首页

https://docs.microsoft.com/en-us/sql/?view=sql-server-ver15

SQL server历史版本文档

https://docs.microsoft.com/en-us/previous-versions/sql/

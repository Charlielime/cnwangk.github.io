我相信有不少小伙伴已经用过eureka，那么问题来了，Nacos是个啥？

看到这个标题，MySQL数据库与Nacos搭建监控服务，它们有什么关系么？

其实是Nacos支持连接MySQL，内部已配置好数据源、连接池供我们使用。如果使用其它数据源（比如信创要求，使用达梦数据库比较多），可以通过插件形式适配，模仿MySQL实现方式。具体如何实现，可参考 Nacos 的 github issues。

Windows 原版镜像下载地址

发表于 2022-06-30 更新于 2023-05-15 分类于 实用工具 本文字数： 1.2k 阅读时长 ≈ 4 分钟

搜集了部分个人使用过的 Windows 原版镜像地址，均来自网络资源。

注意：目前不建议立即升Windows 22H2，可以使用 VMware 体验。

tips：建议使用 BT 工具下载，效果更佳。

business editions：商业多版本选择（包含企业版、专业版、家庭版等…），一般而言我们选择下载这个版本；下载对应自己平台（比如 x64 或者 ARM64 ），这里只提供 64 位版本链接地址。

简单科普，关于镜像命名名称，如何解读？

zh-cn_windows_11_business_editions_version_22h2_x64_dvd_914b57a5.iso

  • zh-cn：通常代表有简体中文支持；
  • windows_11：表示 Windows 操作系统版本第 11 代；
  • business_editions_version：商业版本（包含企业版、专业版、家庭版等…）；
  • 22h2：可以看做是发行版的版本代号；
  • x64_dvd：64 位操作系统，DVD（光盘镜像）形式发售。

以前可视化运维管理界面应用程序在项目rocketmq-externals中rocketmq-console目录下：

https://github.com/apache/rocketmq-externals/tree/master/rocketmq-console

后面再次使用时才发现仓库改成rocketmq-dashboard了，已经找不到rocketmq-console。
